The Joplin desktop app is available for Windows, macOS, and Linux. It provides a full-featured graphical interface for managing your notes, notebooks, and tags, with support for Markdown editing, plugins, and synchronisation.

## Installation

To download and install the desktop app for your operating system, see the [Installation](/installation) page.

## Getting started

When you first open Joplin, you will see three main panels:

- **Notebooks panel** (left) — lists your notebooks and sub-notebooks.
- **Note list** (centre) — shows the notes in the selected notebook.
- **Note editor** (right) — displays and edits the currently selected note.

<Steps>
  <Step title="Create a notebook">
    Right-click the notebooks panel and select **New notebook**, or use the **File** menu. Notebooks help you organise notes by topic or project.
  </Step>
  <Step title="Create a note">
    Press **Ctrl+N** (Windows/Linux) or **Cmd+N** (macOS), or click the **New note** button in the toolbar. Notes are written in Markdown.
  </Step>
  <Step title="Write in the editor">
    Type Markdown in the left half of the editor. The right half shows a live rendered preview. You can also switch to a WYSIWYG editor from **View > Toggle editors**.
  </Step>
  <Step title="Organise with tags">
    Add tags to a note from the note properties panel (click the information icon at the top of the editor). Tags let you cross-reference notes across different notebooks.
  </Step>
  <Step title="Set up synchronisation (optional)">
    Go to **Tools > Options > Synchronisation** and choose a sync target — Joplin Cloud, Nextcloud, Dropbox, OneDrive, WebDAV, or a local directory. See the [Sync overview](/sync/overview) page for full instructions.
  </Step>
</Steps>

## Key features

<CardGroup cols={2}>
  <Card title="Markdown editing" icon="pen-to-square">
    Write notes in Markdown with a live side-by-side preview, or use the rich-text WYSIWYG editor. Both modes are always available.
  </Card>
  <Card title="Notebooks and tags" icon="folder">
    Organise notes into nested notebooks and apply multiple tags per note for flexible cross-referencing.
  </Card>
  <Card title="End-to-end encryption" icon="lock">
    Enable E2EE from **Tools > Options > Encryption** to encrypt all synced data so only you can read it.
  </Card>
  <Card title="Plugin support" icon="puzzle-piece">
    Extend Joplin with community plugins. Browse and install them from **Tools > Options > Plugins**.
  </Card>
  <Card title="Web Clipper" icon="scissors">
    Save web pages and screenshots directly into Joplin using the browser extension. See [Web Clipper](/apps/web-clipper).
  </Card>
  <Card title="Evernote import" icon="file-import">
    Import notebooks from Evernote as ENEX files — including formatted content, attachments, and metadata — via **File > Import**.
  </Card>
</CardGroup>
